Package: com.homeaway.devtools.jenkins.testing

[Groovy] Class InvalidlyNamedScriptWrapper

        • com.homeaway.devtools.jenkins.testing.InvalidlyNamedScriptWrapper

        • Wraps a Script whose filename would not be a valid JVM class name.

          Redirects all Script interface methods to the wrapped Script, calling them indirectly to prevent their names from causing problems with Java classloaders.
