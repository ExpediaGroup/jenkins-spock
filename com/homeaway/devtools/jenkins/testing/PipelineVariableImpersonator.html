Package: com.homeaway.devtools.jenkins.testing

[Groovy] Class PipelineVariableImpersonator

    • com.homeaway.devtools.jenkins.testing.PipelineVariableImpersonator

    • Takes the place of a GlobalVariable in a JenkinsPipelineSpecification context. Intercepts every method call and forwards it to a mock for the current test-run. Mocks are named with the pattern GlobalVariableName.methodName - for example, in the following pipeline script under test: 

      
SomeVariable.someFunction("foo")
      The someFunction call could be verified by 
      
then:
	1 * getPipelineMock("SomeVariable.someFunction")("foo")

      awitt
